Agent Sandbox Attack Research

The project has converged on one main paper direction: a safe NDSS-style benchmark for semantic sandbox escape in AI coding agents.

Selected Thesis

Agent sandboxes are usually evaluated as OS isolation boundaries, but coding agents also cross a semantic boundary through repository files, configuration, tool manifests, and startup state. SSE-Bench will measure this boundary directly.

Why This Direction

5/5

Novelty: fills the gap between prompt-injection benchmarks and container escape benchmarks.

5/5

Feasibility: safe toy harness and sentinel artifacts allow quick iteration.

NDSS

Fit: security measurement, threat model, reusable artifact, decomposed attack metrics.
